# Funcion Color en cssnext

## Traduce el color a rgb a es decir rgb + alpha 

### Tipos de uso 

#### Mas Claro 
*	Alpha ->  background: color ( red alpha(-10%) )

#### Mas oscuro 
*	blackness ->  background: color ( red blackness(-10%) )

#### Saturacion 
*	lightness ->  background: color ( red lightness(-10%) )
